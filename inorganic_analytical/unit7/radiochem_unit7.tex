\section{Purpose}
\ce{KF}板からの放射線を、板の線密度を変えながらGM管で測定することによって、\ce{KF}の吸収係数$\lambda$を求める。

\section{Experimental}
\begin{enumerate}
    \item GM管で、測定電圧はUnit.1の実験で決定した1100 Vとして、バックグラウンドを測定した。
    \item \ce{^{14}C}線源をGMスタンドの上から4段目にセットし、放射線を測定した。
    \item 様々な厚みの\ce{KF}板を線源の上におき、再び放射線を測定した。
\end{enumerate}

\section{Discussion}
\subsection{課題1}
$C(x)$を$0 \ge x \ge d$の範囲で積分し、
\begin{equation}
    I(x) = \int_0^d \dx{x}C(x) = \frac{C_0}{\lambda}\lr{1-e^{-\lambda d}}
\end{equation}
となる。

\subsection{課題2}
式(4.1)にfittingを行うと、$\lambda = 6.0 $ cm$^2$/g となる。また、飽和計数$C_\infty = 114.7$ cpmとなる。

\subsection{課題3}
実効的に、$\frac{S}{\lambda W_{\text{KF}}}$ molの\ce{KF}から放出される放射線を考えれば良いことになる。\ce{^40 K}の崩壊定数は$\frac{\ln{2}}{{T_{1/2}}}\eta$であり、
\begin{equation}
    I_{\text{eff}} = \frac{\ln{2}}{T_{1/2}} \frac{S N_A R_{\text{K-40}}}{\lambda W_{\text{KF}}}\eta
\end{equation}


\subsection{課題4}
実測値を用いると、$I_{\text{eff}} = 25.50$ Bqとなる。また、飽和状態での計数は114.7 cpm = 1.912 Bqであるので
\begin{equation}
    \varepsilon = \frac{1.912}{114.7} = 7.50 \%
\end{equation}
となる。
